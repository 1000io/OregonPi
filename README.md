OregonPi
========

Sensor Oregon Scientific decoder for Raspberry Pi


Install
-------

```sudo apt-get install git-core```
```sudo apt-get update```
```sudo apt-get upgrade```
```git clone git://git.drogon.net/wiringPi```
```cd wiringPi```
```git pull origin```
```./build```

cd ..
git clone git://github.com/1000io/OregonPi
cd OregonPi
make

Execute
-------

sudo ./test
